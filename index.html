Инструкция по эксплуатации

Эмулятор советского микрокалькулятора "Искра-123"
Для включения калькулятора нажмите клавишу "СК".

- Режимы работы: программируемый (П) и режим ручного счёта и вычисления по программе (Р).
- Регистры: Стек из 3 регистров для работы (вычислений) и регистры A1-A5 для хранения чисел в памяти.

Режимы работы переключаются с помощью двухклавишного переключателя П-Р.
Точность вычислений задаётся с помощью клавиш точности (В (Выключена, т.е. точность до 16 разрядов целой части и до 15 дробной разрядной сетки дисплея), 13 знаков после запятой, 11 знаков, 9, 7, 5, 3).

Переполнение разрядной сетки дисплея: признак - у всех разрядов загорятся запятые! Для продолжения работы после переполнения необходимо нажать клавишу "СК". Переполнение наступает при количестве разрядов целой части числа более 16.

Режим ручного счёта и вычисления по программе (Р):
В этом режиме калькулятор работает как обычный арифметический калькулятор, позволяя выполнять операции над числами, в том числе хранящимися в регистрах A1-A5
Для выполнения счёта по программе необходимо:
1) Ввести программу в программируемом режиме ("П").
2) Нажать клавишу "Р" на переключателе режимов.
3) Нажать клавишу "КП", чтобы программа выполнялась с начала.
4) Ввести число, если необходимо (если предполагается начальный ввод в программу)
5) Нажать клавишу "Пуск".
Результат выполнения программы вычислений отобразится на дисплее.

Программируемый режим (П):
В этом режиме задаётся программа, путём последовательного нажатия необходимых клавиш, нажатие которых последовательно будет имитироваться при выполнении программы. Для пошагового просмотра программы и замены (при необходимости) отдельных команд используются клавиши "Пуск" и "СК". Для перехода к просмотру следующей команды необходимо нажимать клавишу "Пуск", а для перехода к предыдущей - клавишу "СК". Для замены текущей команды нажмите клавишу "СК", на экране погаснет старая команда, а затем нажмите необходимую клавишу (новую команду), которая запишется в программу вместо погасшей раннее на экране старой команды. Для возврата в начало программы перейдите в режим "Р" и нажмите клавишу "КП", а затем вернитесь в режим "П".
Ввод программы обязательно завершается командой "КП".

Назначение клавиш:
0-9, ",": Ввод цифр и запятой
А1-А5: Сохранение числа в регистре памяти A1-A5 и загрузка из него.
Примечание: если перед клавишей А1-А5 следовала одна из: ввод числа, "=", "↑", "/", "/-/", "|x|", ")", "√", то происходит сохранение числа в регистр. Если же перед клавишей А1-А5 следовала одна из: "+", "-", "х", "÷", "¬÷", " ②", " ③", A1-A5, "♦", "T", "СК", " ►", "КП", "СТОП", то происходит загрузка числа из регистра памяти в стек для дальнейших операций.
АВТОВЫБОРКА ИЗ РЕГИСТРА А1 ЭМУЛЯТОРОМ НЕ ПОДДЕРЖИВАЕТСЯ, В ОТЛИЧИЕ ОТ ОРИГИНАЛЬНОГО УСТРОЙСТВА!
СК:

  • В режиме "Р": если перед этим была нажата клавиша "=", то происходит полная очистка всех регистров и сброс всех операций. Регистры памяти А1-А5 НЕ ЗАТРАГИВАЮТСЯ!
    Если же перед этим была нажата любая другая клавиша, то происходит только очищение текущего вводимого значения и экрана, можно заново ввести текущее значение.
  • В режиме "П": позволяет просматривать программу пошагово с конца (показывает предыдущую команду, смещает текущую позицию (выполняемую команду) назад).
②: добавить число к регистру А2
③: добавить число к регистру А3
+, -, х, ÷, ¬÷: клавиши операций сложения, вычитания, умножения деления, обратного деления
√: квадратный корень
↑: возведение в квадрат
|x|: модуль числа
/: целая часть числа
=: выполнить операцию (равно)
/-/: смена знака числа
♦: вывести содержимое регистров памяти (НЕ ПОДДЕРЖИВАЕТСЯ!)
►: перейти к команде, перед которой в программе стоит команда "метка" (Т)
T: пометить команду меткой (ставить перед текущей командой)
КП: конец программы
Примечание: в режиме "Р" возвращает программу в начало.
ПУСК:
  • В режиме "Р": запускает счёт по программе с текущей позиции (если нажать "КП" перед этим - программа начнёт выполняться с начала)
  • В режиме "П": смещает вперёд текущую позицию в программе (т.е. отображает следующую команду).
СТОП: позволяет вставить в программу прерывание для ввода очередного значения и отображения промежуточного результата вычислений (НЕ ПОДДЕРЖИВАЕТСЯ!)
(, ): скобки.
